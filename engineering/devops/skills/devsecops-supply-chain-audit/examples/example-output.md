## Executive Summary

Three ecosystems scanned. npm has two HIGH CVEs in direct production dependencies (both prototype-pollution family) and one known-exploited CVE in a transitive dev dep (upgrading `vite` fixes it). Docker images pin by tag not digest. Terraform providers are unpinned in two modules. Branch protection is enabled on `main` but required reviewers is 0 — any maintainer can self-merge. 3. No CODEOWNERS — infra and workflows unprotected *(SEC-007)*
